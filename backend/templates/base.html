How is it working?

Very simple.

Let's assume we have a crate and lowest Steam price of that crate is $5.
Count of items in the crate which more expensive than $5 is X
Count of items which cheaper than $5 is Y
If X => Y than better to open the crate, right?

This application, collects crate data and groups items by their Steam price. So you can see the possibility of the item being more expensive than the crate.

Why?

This is a hobby project. It helps me to decide if i should open crate or sell it. I think, maybe there are other players who wants to see possibilities about their crate and i can help them.

This application does not provide investment advice. It was not considered key price if crate needs a key to open it.
Crate data are not real time. The data are updated periodically. Data are collected from pubgshowcase.com
